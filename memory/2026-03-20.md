## Session: UHNW Upgrade & Real Company Names Fix

### Problem Identified
User requested:
1. **Real company names** - Lambda was showing generic placeholder names (CONSORCIO PERU EIRL, etc.) instead of actual SUNAT company names
2. **UHNW/High-end design** - Certificate and admin dashboard needed luxury exclusive aesthetic
3. **More admin dashboard features** - Enhanced functionality beyond basic verification

### Solutions Implemented

#### 1. Real Company Names via Decolecta
- **Fix**: Modified `get_sunat_data_real()` to properly extract `nombre_o_razon_social` from Decolecta response
- **Logic**: 
  - First tries Decolecta API for real SUNAT data
  - If "No encontrado", returns `None` (no mock fallback for name)
  - OSCE scraper runs separately for sanctions
  - If no SUNAT data found: shows "⚠️ RUC NO ENCONTRADO EN SUNAT"
- **Test result**: RUC 20611944668 → "ZAMORA JARA INGENIEROS SAC" ✅

#### 2. UHNW Lambda Deployment
- **File**: `lambda_function_uhnw.py` → deployed as v7
- **Features**:
  - Real SUNAT names only (no placeholder generation)
  - Ethical OSCE scraper with 30min cache, 24h per-RUC cache
  - 5s rate limiting between requests
  - User-Agent: `ConflictZero-Bot/1.0 (contacto@czperu.com)`
  - Better scoring: -50 for RUC not found, -30 per inhabilitación vigente
  - Professional HTML certificate with gold/navy theme

#### 3. UHNW Admin Dashboard
- **URL**: `https://conflictzero-certificados-prod.s3.amazonaws.com/admin-uhnw.html`
- **Design elements**:
  - Typography: Playfair Display + Inter
  - Color palette: Navy (#0d1b2a) + Gold (#c9a227)
  - Glassmorphism header with blur
  - Animated score ring with CSS conic-gradient
  - Hover effects on stat cards
  - Responsive grid layout
- **Features**:
  - Real-time verification
  - Stats cards (Verificaciones hoy, Certificados, Alertas, Base total)
  - Animated score display with risk levels
  - Sanctions list with entity badges
  - Certificate generation button
  - Clean/minimal navigation

#### 4. Certificate UHNW Design
- **Template**: Professional HTML with watermark
- **Sections**:
  - Gold header bar
  - Logo with CZ watermark
  - Circular score display
  - Company info grid
  - Data source verification (SUNAT/OSCE/TCE badges)

  - Legal disclaimer
