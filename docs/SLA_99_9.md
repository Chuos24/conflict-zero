# SLA 99.9% - Service Level Agreement

## Garantía de Disponibilidad

Conflict Zero garantiza un **99.9% de uptime mensual**, equivalente a:
- **Máximo 43 minutos de downtime permitido por mes**
- **Máximo 8.76 horas de downtime permitido por año**

## Métricas de Monitoreo

| Métrica | Valor | Descripción |
|---------|-------|-------------|
| Uptime garantizado | 99.9% | Tiempo de operación mensual |
| Latencia p95 | < 500ms | 95% de requests responden en menos de 500ms |
| Health check | Cada 30 min | Verificación automática de servicios |
| Auto-restart | < 60s | Reinicio automático en caso de falla |

## Endpoints de Monitoreo

### Health Check
```
GET https://conflict-zero-api.onrender.com/api/v1/health
```

Respuesta:
```json
{
  "status": "healthy",
  "version": "1.0.0",
  "timestamp": "2026-03-27T03:30:00Z",
  "services": {
    "database": "up",
    "redis": "up"
  },
  "sla": {
    "uptime_seconds": 86400,
    "uptime_percentage": 99.95,
    "guarantee": "99.9%"
  }
}
```

### Status Page Pública
```
GET https://conflict-zero-api.onrender.com/api/v1/status
```

Muestra estado actual de todos los componentes.

## Componentes Críticos

| Componente | Impacto | Estado Requerido |
|------------|---------|------------------|
| API REST | Crítico | 99.9% uptime |
| PostgreSQL | Crítico | 99.9% uptime |
| SUNAT Integration | Core | 99.5% uptime |
| Redis | Performance | 95% uptime (opcional) |

## Compensación por Incumplimiento

Si el uptime mensual cae por debajo de 99.9%:
- **99.0% - 99.9%**: 10% de crédito en próxima factura
- **95.0% - 99.0%**: 25% de crédito en próxima factura
- **< 95.0%**: 50% de crédito + review de arquitectura

## Exclusiones

El SLA no cubre:
- Mantenimiento programado (con 24h de aviso)
- Fuerza mayor (ataques DDoS, fallas de ISP)
- Uso incorrecto de la API por parte del cliente
- Límites de plan excedidos
